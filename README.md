# LXHistory_Sync

> 基于 Plasmo 框架的 Chrome 浏览器历史记录同步工具，支持 WebDAV 协议，实现跨设备数据备份与恢复

---

## ✨ 核心特性

### 🔐 隐私优先
- 所有数据仅在本地和您的 WebDAV 服务器之间传输
- 支持多种端到端加密方式，确保数据安全
- 加密密钥仅存储在浏览器本地，不会上传到任何服务器

### 🔄 智能同步
- 双向同步：支持上传历史记录到云端，或从云端恢复到本地
- 自动合并：智能合并本地和云端的历史记录，避免数据冲突
- 灵活配置：默认同步最近 1000 条记录，可根据需求调整

### 🎨 个性化体验
- 主题切换：支持亮色/暗色/自动三种主题模式
- 域名分组：按域名组织历史记录，便于浏览
- 搜索功能：快速定位所需的历史记录
- 多样化图标：支持字节跳动、Google、百度、字母等多种图标源
- 自定义样式：多种复选框样式可选

### 🔒 强大加密
- **AES-256-GCM**：推荐使用，提供认证加密
- **AES-256-CBC**：经典加密方式
- **AES-256-CTR**：流加密方式
- **ChaCha20-Poly1305**：高性能加密方式

## 🚀 快速开始

### 安装扩展

#### 从源码构建

```bash
# 1. 克隆项目
git clone <repository-url>
cd LXHistory_Sync

# 2. 安装依赖
npm install

# 3. 构建扩展
npm run build

# 4. 加载到 Chrome
# 打开 chrome://extensions/
# 开启"开发者模式"
# 点击"加载已解压的扩展程序"，选择 build/chrome-mv3-prod 目录
```

#### 使用预构建包

下载预构建的扩展包，拖动到 `chrome://extensions/` 页面即可安装

## 📖 使用指南

### 第一步：配置 WebDAV

1. 点击浏览器工具栏中的 LXHistory_Sync 图标
2. 进入设置页面，切换到「WebDAV 设置」标签
3. 填写服务器信息：
   - WebDAV 服务器地址
   - 用户名
   - 密码
4. （可选）启用加密：
   - 选择加密类型
   - 设置加密密钥
5. 点击「保存配置」

### 第二步：个性化设置

在「通用设置」标签中配置：

| 选项 | 说明 |
|------|------|
| 显示搜索框 | 是否在主界面显示搜索框 |
| 复选框样式 | 选择复选框的外观样式 |
| 按域名分组 | 是否按域名分组显示历史记录 |
| 显示 URL | 是否显示完整的 URL 地址 |
| 图标来源 | 选择网站图标的获取方式 |

在「主题设置」标签中选择：
- **自动**：跟随系统主题
- **亮色主题**：始终使用亮色模式
- **暗色主题**：始终使用暗色模式

### 第三步：同步历史记录

#### 上传到云端
```
点击扩展图标 → 点击「同步到云端」 → 等待完成
```

#### 从云端恢复
```
点击扩展图标 → 点击「从云端同步」 → 等待完成
```

### 第四步：浏览历史记录

- 在扩展弹出窗口中查看本地历史记录
- 使用搜索框快速过滤记录
- 点击记录项在新标签页中打开网址
- 按域名分组浏览，更清晰直观

## ⚠️ 重要提示

### 安全建议
- WebDAV 密码和加密密钥存储在浏览器本地，请确保设备安全
- 启用加密后，请妥善保管加密密钥，丢失将无法解密云端数据
- 建议使用强密码保护您的 WebDAV 账户

### 使用建议
- 同步操作需要稳定的网络连接
- 部分服务器可能有存储空间或请求频率限制
- 建议根据需要手动同步，避免过于频繁的操作
- 首次同步建议在稳定的网络环境下进行

## 🔧 故障排除

### 常见问题

| 错误信息 | 可能原因 | 解决方法 |
|----------|----------|----------|
| WebDAV 配置未设置 | 未完成服务器配置 | 进入设置页面填写 WebDAV 信息 |
| 无效的用户名或密码 | 凭据错误 | 检查用户名和密码是否正确 |
| 无法连接到 WebDAV 服务器 | 网络或地址问题 | 检查网络连接和服务器地址 |
| 没有权限访问 WebDAV 服务器 | 权限不足 | 检查服务器权限设置 |
| WebDAV 服务器内部错误 | 服务器问题 | 联系服务器管理员 |
| 解密失败 | 密钥或类型错误 | 检查加密密钥和类型是否一致 |

### 查看日志

扩展的错误和操作日志会显示在浏览器开发者工具中：

```
打开扩展弹出窗口 → 按 F12 → 切换到"控制台"标签页
```
